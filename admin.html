LOCAL BACKUP & TOOLS SAFE EDITING


      

      

        The admin will try to load drive-card.json &
        drive-precons.json, and fall back to
        localStorage backups if needed.
      


      

        
Workflow suggestion:

        
    
          
  1. Open this admin and let it load drive-card.json and
            drive-precons.json.
          
    2. 
          
  2. Use Single and Bulk tabs to edit or create cards.
    3. 
          
  3. Check precons using the Precons tab to verify deck contents.
    4. 
          
  4. Click Export Card JSON and Export Precon JSON
            to download updated files.
    5. 
          
  5. Replace the JSON files in your repo with the exported ones and commit.
    6.
